This folder holds any image files that are referenced in the quiz.
The name ImageDatabase of this folder is a placeholder. It can be changed to suit your study.

This directory is where the login screen will automatically direct the user, serving as the default location.
By placing the quiz's image files in this folder, it eliminates the need for the user to navigate to a different directory.


See Documentation https://baines-imaging-research-laboratory.github.io/ImageQuizzerDocumentation/
